If you subscribe to a S or BS you will be able to get the current value or initial value
You have to define a default value whenever you declare Behavior Subject or Subject based upon the data type
In Subject, each next subscriber receives only the upcoming values. In Behavior Subject,each next subscriber receives one previous value and upcoming values.
